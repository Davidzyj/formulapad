# FormulaPad Usage Guide

## Calculate

1. Open the Calculate tab.
2. Enter a formula such as `20000 * (1 + 0.035)^3`.
3. Tap Calculate.
4. The result appears and the calculation is saved to History.

Supported examples:

- `128 * 0.85`
- `sqrt(144)`
- `sin(30)`
- `price = 299; count = 12; price * count`

Use the angle segmented control to switch between degrees and radians.

## History

- Tap Reuse to send an old expression back to Calculate.
- Tap the star to mark an entry as favorite.
- Delete individual entries or clear all history.
- Free mode keeps the latest 20 history entries.

## Notes

FormulaPad Pro unlocks math notes.

1. Calculate a formula.
2. Tap Save Note.
3. Enter a title, category, and optional remarks.
4. Save.
5. Open Notes to edit, favorite, or delete the note.

## Templates

Open Tools, then Templates.

- Free templates: percentage, discount, compound interest, BMI.
- Pro templates: loan payment, average, triangle area, speed.

Each template has fields, a result, an explanation, and a Save to History action.

## Unit Conversion

Open Tools, then Convert.

1. Choose a category.
2. Enter a value.
3. Choose source and target units.
4. Save the conversion to History if needed.

Supported categories: length, weight, area, volume, temperature, time, and speed.

## Function Plot

FormulaPad Pro unlocks function plotting.

1. Open Tools, then Plot.
2. Enter a function using `x`, such as `x^2` or `sin(x)`.
3. Tap Plot.
4. Save the plot expression to History if needed.

## Settings

- Switch language between automatic, English, Simplified Chinese, and Japanese.
- Unlock or restore FormulaPad Pro.
- Open Privacy Policy, Support, or Contact Support.
