# FormulaPad Handoff

## Summary

FormulaPad is a SwiftUI iPhone-only app built from an empty directory. It is a local-first math notebook calculator with one-time Pro unlock.

## Key Product Decisions

- No custom backend.
- No accounts.
- No ads, analytics, or tracking.
- Local JSON persistence in Application Support.
- StoreKit 2 non-consumable purchase for Pro.
- App UI forces light mode for review-safe readability.
- App supports English, Simplified Chinese, and Japanese.

## Important Paths

- Xcode project: `FormulaPad.xcodeproj`
- App source: `FormulaPad/`
- Main app entry: `FormulaPad/App/FormulaPadApp.swift`
- State store: `FormulaPad/Services/FormulaPadStore.swift`
- Expression parser: `FormulaPad/Services/ExpressionEvaluator.swift`
- Templates: `FormulaPad/Services/TemplateCatalog.swift`
- Units: `FormulaPad/Services/UnitCatalog.swift`
- Purchase manager: `FormulaPad/Services/PurchaseManager.swift`
- Info.plist: `FormulaPad/Resources/Info.plist`
- App icon: `FormulaPad/Resources/Assets.xcassets/AppIcon.appiconset/FormulaPadIcon.png`
- StoreKit config: `FormulaPad/Resources/StoreKit/FormulaPad.storekit`
- GitHub Pages pages: `docs/index.html`, `docs/privacy.html`, `docs/support.html`

## Regenerating Project

The Xcode project is generated by:

```sh
ruby scripts/generate_xcodeproj.rb
```

Run it after adding/removing source files.

## Localization

UI strings are controlled by `FormulaPad/Services/LocalizationService.swift` so the app can switch languages immediately from Settings.

`InfoPlist.strings` files localize `CFBundleDisplayName`:

- English: FormulaPad
- Simplified Chinese: 算记
- Japanese: 数式メモ

## Persistence Notes

`FormulaPadStore` publishes arrays/dictionaries and updates them by assigning new values after mutation. This is intentional for SwiftUI refresh reliability.

Stored state:

- History
- Notes
- Variables
- Angle mode
- Language preference

## Review-Safety Notes

- `UIUserInterfaceStyle` is set to `Light`.
- SwiftUI root also uses `.preferredColorScheme(.light)`.
- Text, placeholder, disabled, and secondary colors are explicit.
- Settings does not display raw support URL, privacy URL, email, or Bundle ID.
- `ITSAppUsesNonExemptEncryption` is `false` in the target Info.plist.
- Target build setting confirms `INFOPLIST_FILE = FormulaPad/Resources/Info.plist`.
- `TARGETED_DEVICE_FAMILY = 1`.
