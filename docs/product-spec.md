# FormulaPad Product Spec

## Product Positioning

FormulaPad is a local-first math notebook calculator for iPhone. It combines natural formula input, scientific calculation, reusable history, notes, templates, unit conversion, simple function plotting, and a one-time Pro unlock.

The app does not require an account, does not use a custom backend, does not run ads, and does not initiate network requests except when the user explicitly opens App Store purchase flows or external privacy/support pages.

## Naming

- English app name: FormulaPad
- Simplified Chinese display name: 算记
- Japanese display name: 数式メモ
- Bundle ID: com.zhouyajie.formulapad
- Version: 1.0.0
- Monetization: one-time in-app purchase, FormulaPad Pro
- Product ID: com.zhouyajie.formulapad.pro

## Target Users

- Students who need a clean calculator with reusable formulas and notes.
- Teachers and parents who want readable calculation steps and sharable results.
- Engineers and productivity users who reuse formulas, variables, unit conversions, and quick plots.

## V1 Scope

### Free Features

- Standard calculator and natural expression input.
- Basic scientific functions.
- Recent calculation history, limited to 20 saved entries.
- A small set of built-in templates.
- Basic unit conversion.
- Local-only storage.
- Light mode UI with explicit readable colors.

### Pro One-Time Unlock

- Unlimited calculation history.
- Math notes with title, category, remarks, and favorite status.
- Custom reusable formulas.
- Full built-in template library.
- Simple function plotting.
- Export/share calculation text.
- Extra visual themes and app icon variants are planned, but not required for 1.0.0.

## Not In V1

- AI solving.
- Photo math recognition.
- User accounts.
- Custom backend.
- Ads or tracking.
- Cloud sync.
- App Store screenshots.
